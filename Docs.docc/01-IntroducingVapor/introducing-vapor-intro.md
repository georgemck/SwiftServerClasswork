Learn the fundamentals of server-side Swift by creating your first HTTP server application from scratch.

## Objectives

By the end of this lab you will understand how to:

- Create a basic server-side Swift project using modern frameworks.
- Explain the HTTP request-response cycle and server architecture.
- Create a simple HTTP GET endpoint with proper routing and query parameters.
- Demonstrate testing server endpoints using command-line tools.

## Prerequisites

- Basic Swift programming knowledge including functions, structs, and error handling.
- Familiarity with HTTP concepts such as requests, responses, and status codes.
- Understanding of command-line tools for testing web services.
- Set up VS Code with the Swift extension and ensure Swift is installed on your device.
