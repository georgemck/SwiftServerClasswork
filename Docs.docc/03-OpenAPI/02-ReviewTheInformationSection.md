Step 2

## Instructions

Update your `openapi.yaml` file as shown below:

```yaml
---
openapi: 3.1.1
info:
  title: PollsApp
  version: 0.1.0
```

The `info` section provides metadata about the API, such as its title and version.

At this point, the configuration sets the OpenAPI version to `3.1.1`, the API title to `PollsApp`, and the current
version to `0.1.0`.
