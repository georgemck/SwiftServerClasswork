Step 1

## Instructions

Begin by creating the OpenAPI file. Add a new file named `openapi.yaml` to the `Sources/PollsApp` directory. You build
the file gradually as you progress through these steps.

Add the following to your `openapi.yaml` file:

```yaml
---
openapi: 3.1.1
```

This is the starting point of the specification and indicates the OpenAPI version being used. In this case, the version
is `3.1.1`.
