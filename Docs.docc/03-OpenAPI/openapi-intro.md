Use OpenAPI specifications to design, document, and build type-safe APIs for your Swift Server applications.

## Objectives

OpenAPI is a common format used by many software teams to create and ingest API documents. By the end of this lab, you
will understand how to:

- Create an OpenAPI document.
- Create a `schema component` for polls.

## What is OpenAPI?

[OpenAPI][1] is an API description standard that provides three benefits:

- Commonality of use and support by tool vendors, meaning flexibility in tool vendor selection.
- Commonality of knowledge by developers and engineers, meaning flexibility in staffing.
- Abstraction of API description, facilitating adoption of innovations in API behaviors while avoiding total rewrites.

[1]: https://www.openapis.org
