Build endpoints for updating and deleting polls.

## Objectives

In this chapter, you will extend the Create and Read Polls functionality that you built in the previous chapter by
adding the ability to update and delete polls. By the end of this chapter, you will understand how to:

- Create a `PATCH` endpoint in OpenAPI for updating a poll.
- Create a `PATCH` endpoint with Swift OpenAPI.
- Create a `DELETE` endpoint in OpenAPI for deleting a poll.
- Create a `DELETE` endpoint with Swift OpenAPI.

## RESTful resources

In the last chapter, you received a briefing on RESTful resources. This chapter completes the endpoints necessary for
creating, reading, updating, and deleting a server resource.
