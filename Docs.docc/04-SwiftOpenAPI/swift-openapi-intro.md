Learn to use autobuilt code from Swift OpenAPI Generator.

## Objectives

By the end of this lab, you will understand how to:

- Examine types autogenerated from Swift OpenAPI Generator.
- Create an `APIHandler` that conforms to the `APIProtocol` protocol.
- Demonstrate testing your API handler with Swift Testing.

## What is Swift OpenAPI Generator?

[Swift OpenAPI Generator](https://github.com/apple/swift-openapi-generator) transforms an `openapi.yaml` specification
into generated types. You can then use these types to create handler functions for your API.

Swift OpenAPI Generator takes advantage of the language's strong type system, protocol-oriented architecture, and safety
features. The generated code integrates seamlessly with popular Swift Server frameworks such as Vapor and also works
with Swift's concurrency model.

> Note: The Swift OpenAPI Generator is a Swift Package Manager (SPM) plugin.
