Learn server-side Swift development, covering modern web frameworks, APIs, and deployment strategies for building
production-ready applications.

## Learning objectives

Swift Server Fundamentals teaches you to build production-ready Swift APIs using modern frameworks and tooling. You
create server applications with common libraries and test-driven development to solve common server-side problems.

Follow along during the code-along sessions by typing code directly on your computer.

## Prerequisites

This workshop targets developers familiar with Swift, developer technologies such as Git and GitHub, agentic tooling
such as Claude Code, and server technologies such as containers.

Ensure you have the latest macOS and Swift installed with an editor such as Xcode, VSCode, or Neovim ready for
development.
