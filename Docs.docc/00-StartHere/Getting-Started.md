Set up your server-side Swift development environment and verify it works correctly before starting the workshop.

## Prerequisites

You need these tools and skills to complete this workshop successfully:

- Swift installed on your system (Swiftly is recommended for version management).
- Xcode 16 or later, Visual Studio Code with Swift extensions, or a similar editing environment.
- Familiarity with terminal commands for file management and navigation.
- Swift programming fundamentals (consider taking Swift Language Fundamentals if you need foundational knowledge).

### Check prerequisites

Verify your Swift version meets the workshop requirements:

```bash
swift --version
```

This command confirms you have a compatible Swift version installed.
